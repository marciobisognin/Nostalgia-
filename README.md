# 📺 NostalgiaOS 95

Uma página nostálgica no estilo **desktop de computador dos anos 90** (Windows 95/98)
com um retrospecto dos melhores desenhos animados e séries das décadas de **70, 80, 90 e 2000**.

Cada card traz:

- 🎬 **Como foi criada** a atração
- 🏁 **Como terminou**
- 🚶 **Por onde andam** os atores e criadores
- ▶ Um botão **Assistir no YouTube** com a busca pronta pelo filme/episódio completo

## 🌐 Como ver online (GitHub Pages)

1. Vá em **Settings → Pages** neste repositório.
2. Em **Build and deployment → Source**, escolha **Deploy from a branch**.
3. Selecione a branch `claude/nostalgic-90s-github-pages-e640j4` (ou faça o merge na `main` e use `main`) e a pasta `/ (root)`.
4. Salve. Em alguns minutos o site estará em:
   `https://<seu-usuario>.github.io/Nostalgia-/`

## 🖱️ Como usar

- Clique nas **abas 📁 70s / 80s / 90s / 2000s** ou no menu **Iniciar** para navegar entre as décadas.
- Cada card abre a busca do título no YouTube ao clicar em **▶ ASSISTIR NO YOUTUBE**.

## ℹ️ Sobre os links do YouTube

Os botões usam **links de busca do YouTube** em vez de IDs de vídeo fixos.
Vídeos avulsos costumam ser removidos por direitos autorais e os links "quebram" com o tempo;
a busca sempre leva ao conteúdo que estiver disponível no momento. É só clicar e apertar PLAY. 🍿
